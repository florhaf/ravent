Troubleshooting Hangouts

We're having trouble connecting with the plugin. Please follow our troubleshooting steps:
To uninstall in Mac OS X:
  1. In Finder, navigate to Applications > Utilities
  2. Open the Terminal application.
  3. Copy and paste the following command into the window:
    /Library/"Application Support"/Google/GoogleVoiceAndVideoUninstaller.app/Contents/MacOS/GoogleVoiceAndVideoUninstaller
  4. Press Enter on your keyboard.
  5. Close all browsers.
  6. Type in your password if prompted. The uninstaller will inform you when it is complete.
  7. Return here (type plus.google.com/hangouts into your browser)
To uninstall in Windows:
  1. Click Start > Settings > Control Panel.
  2. Double-click Add or Remove Programs.
  3. Scroll down to find Google Talk plugin.
  4. Click Remove, then Yes, then Finish.
  5. Refresh page.
    Refresh
To uninstall in Linux:
  1. Go to Applications > Debian > Applications > System > Package Management > Synaptic Package Manager.
  2. Search for "Google Talk Plugin" in the list of packages.
  3. Select the "Google Talk Plugin package" checkbox.
  4. To completely uninstall (including configuration files), select "Mark for Complete Removal." To uninstall while saving your preferences, select "Mark for Removal."
  5. Click Apply.
  6. Select Google Talk Plugin from the "To be completely removed" menu.
  7. Click Apply.
  8. Refresh page.
    Refresh
